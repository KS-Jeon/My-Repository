# Scripts & IaC Study Repository

이 리포지토리는 **실무 운영 과정과 학습 과정에서 반복적으로 발생하는 작업을 효율화**하기 위해  
직접 작성한 **운영 스크립트(scripts)** 와, 인프라 구성의 **일관성과 재현성을 확보하기 위한 IaC(iac)** 를 함께 정리한 포트폴리오입니다.

- `scripts/` : 일상 운영/환경 준비/점검 자동화를 위한 유틸 스크립트
- `iac/` : Terraform/CloudFormation/Bicep 등 IaC 도구를 활용하여 클라우드 인프라 리소스를 선언형으로 구성 및 배포하는 코드 기반 인프라 관리 영역


## Why This Repository Exists

실무에서 **반복되는 운영 작업은 사람의 시간이 아닌 코드가 수행해야 한다**는 원칙 아래,  
학습 과정에서 이해한 내용을 **즉시 도구화·자동화**하여 내 것으로 만들기 위해 정리한 레포지토리입니다.

특히 다음 두 가지 목표에 중점을 두었습니다.

1) **효율화**  
   - 환경 설정, 인증, 점검 등 **비생산적인 반복 업무를 스크립트화**
2) **재현성**  
   - IaC를 활용해 누가 실행하든 **동일한 인프라가 만들어지는 구조**

이 레포는 ‘단순 정리’가 아니라,  
실제로 현업에서 바로 사용할 수 있는 구조와 흐름을 만드는 것을 목표로 합니다.
